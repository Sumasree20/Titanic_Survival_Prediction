# Titanic_Survival_Prediction
Developed a machine learning model using Python, Pandas, NumPy, and Scikit-learn to predict passenger survival on the Titanic dataset. Performed data cleaning, EDA, feature engineering, and Logistic Regression model evaluation.
**Technologies Used:** Python, Pandas, NumPy, Scikit-learn, Logistic Regression

**Skills**: Data Cleaning, Data Preprocessing, Exploratory Data Analysis (EDA), Feature Engineering, Machine Learning, Predictive Analytics, Statistical Analysis, Model Evaluation

**Dataset**: Kaggle Titanic Dataset
